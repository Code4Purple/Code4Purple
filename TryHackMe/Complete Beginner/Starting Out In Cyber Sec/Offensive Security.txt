
# Offensive Security

This Career is for you if:

	1. you enjoy understanding how things work
	2. you are analytical
	3. you like thinking out of the box

The most common offensive security job role is a penetration tester. A pentester is an individual
that is legally employed by an organisation to find vulnerabilities in their products. 

A penetration tester usually requires a broad range of knowledge including:
	
	1. web application security
	2. network security
	3. use of programming languages to write various scripts

Most recently, cloud security has also been gaining popularity as various organisations are now shifting
their infranstructure to cloud providers such as AWS and Azure.

It's also possible to have a speciality in one of thses topics, however a broad knowledge is the best way 
to start out.
